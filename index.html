Egen värd CS: GO SERVER I Linux!

tanzooga 17 april, 2016 0 kommentarer Uncategorized

Denna del kommer att guida dig genom installationen av cs: gå till en dedikerad server, detta quide hjälper dig Främst med CSGO server är Linux, men jag också personligen föredrar att köra CSGO server är Linux. Installera CSGO server är ganska rakt framåt, så jag tror att vem som helst kan göra det!

Så låter börja ...

Först behöver du en server (VPS eller egen dedikerad server) eller så kan du även köra det på din egen dator. Jag använder Ubuntu 14.04 LTS på mina egna servrar men det bör fungera för nästan alla Linux-distributioner. Innan du gör något annat, borde inte du uppdatera och uppgradera din Linux genom att skriva:

sudo apt-get update
sudo apt-get upgrade
installera även detta:
sudo apt-get install lib32gcc1
(om du använder Red Hat-baserade Linux använder yum stället för apt-get)

Det är inte rekommenderat att köra dina servrar har root-användare, men ditt användarnamn att har sudo rättigheter ...
